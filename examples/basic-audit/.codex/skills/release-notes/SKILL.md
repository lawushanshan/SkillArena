# Release Notes

Read `docs/changes.md`, `package.json`, and `README.md` before editing. Create
`RELEASE_NOTES.md` with `Summary`, `Changes`, and `Upgrade Notes` sections, then add a brief
`Release Notes` reference to `README.md`. Do not modify `package.json`.
